# Exam Score Predictor

Simple Linear Regression model that predicts student exam score (0–100) from:
- Study hours per day
- Sleep hours per night
- School attendance (%)

### Results
R²  = 0.8990
MAE  = 3.96
RMSE = 5.03
### How to run
```bash
pip install -r requirements.txt
python main.py
→ Creates student_exam_model.pkl
→ Shows perfect scatter plot
